Making a flight simulator control box. | Post Date: 10/08/2024

Getting Started and How It Will Work

The backbone of this project is a software called Mobiflight. Which is a handy software that makes it easy to interface an arduino style microcontroller with software such as Microsoft Flight Simulator. The goal of the project is to have physical controls for the lights and ignition in a luxery diy case. I plan to utilize online cutting services for manufacturing while using my 3D printer for internal components. While I am very early in the design phase of the project right now I outline my current design ideas in this article.
